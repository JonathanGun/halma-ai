## Prerequisite
1. Python 3

## Setup
```
pip install --upgrade -r requirements.txt
```

only for Python 3.8+
```
pip install --pre --extra-index-url=https://kivy.org/downloads/simple kivy[base]
```

## Running
```
python main.py
```
